About Educated Citizen:

Educated Citizen is Whitney Zilton's project for Hackbright Academy, created in spring 2020. It is a full-stack application with a HTML, CSS, Bootstrap, and Javascript frontend, a PostGreSQL database built with SQLAlchemy, and a Python/Flask backend. APIs utilized are OpenSecrets API and VoteSmartAPI.
